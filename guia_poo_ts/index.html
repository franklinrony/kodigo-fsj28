Ejemplos de Programación Orientada a Objetos en TypeScript

Instrucciones:
  • Asegúrate de instalar las dependencias necesarias ejecutando npm install en la terminal.
  • Luego ejecuta npm run dev para iniciar el servidor de desarrollo.
  • Abre la consola del navegador para visualizar los resultados de los ejemplos.

Explicación de los ejercicios resueltos

  1. CabeceraPagina: Clase que representa la cabecera de una página web, permitiendo definir su título, color, fuente y alineación. Incluye métodos para mostrar sus propiedades en consola.
  2. Calculadora: Clase que implementa operaciones matemáticas básicas como suma, resta, multiplicación, división, potencia y factorial. Los resultados se muestran en consola.
  3. Cancion: Clase que modela una canción con título, género y autor. Permite mostrar los datos de la canción en consola.
  4. Cuenta: Clase que simula una cuenta bancaria, permitiendo depositar, retirar y mostrar información de la cuenta, validando operaciones y mostrando mensajes en consola.
  5. Empleado y Persona: Persona es una clase abstracta base. Empleado hereda de Persona y añade la gestión de sueldo. Permite cargar, imprimir sueldo y mostrar datos personales, además de verificar si es mayor de edad.
